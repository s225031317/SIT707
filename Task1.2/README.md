# Task 1.2 Summary

## Software Failure Examples
- Therac-25: Radiation overdose due to software errors
- AT&T Outage: 5 million calls blocked from a simple break statement
- USS Vincennes: UI error led to shooting down civilian aircraft

## Key Terminology
- Error: Human mistake
- Fault/Bug: Flaw in code  
- Failure: Software behaves incorrectly
- Incident: User notices failure

## Testing Levels
- Unit Testing (JUnit)
- Integration Testing
- System Testing (Selenium)
